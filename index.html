STREETDANCE FIT GUIDE

ストリートダンサー向けファッションブランドまとめ

このサイトは、ストリートダンサー向けのファッションブランドをジャンル別にまとめたガイドです。Lock や HipHop、Pop など、ジャンルごとにおすすめのブランドを紹介します。

オンラインで買えるブランドしか紹介していないので、気になったブランドは、ぜひ公式サイトから最新ラインナップをチェックしてみてください。

はじめに

うまいダンサーを見ていると、「あの人の服かっこいいな」「Lockダンサーがよく履いているあの靴、どこのだろう？」と思うことがあります。

でも、いざネットで調べてみても、ストリートダンスとファッションをジャンルごとにまとめている情報はまだ少なく、知りたい答えにたどり着けないことが多くありました。

それなら、自分で調べてまとめてみよう。
実際に履いている人に聞いたり、人づてに教えてもらったり、Instagramにあるストリートダンサーのインタビュー動画などを参考にしながら、ダンサー目線でブランドを集めていく。

そんなきっかけで始めたのが、STREETDANCE FIT GUIDE です。

このサイトでは、Lock、House、Pop、Break、HipHop、Waack、Krumpなど、ジャンルごとにブランドやアイテムを紹介していきます。

LockとPopはお互いのブランドをチェックすることを強くお勧めします。この2つのジャンル向けに商品を作っているブランドが多いので要チェックです。

まだ未完成な部分もたくさんありますが、これから少しずつ更新していく予定です。

「このブランドもよく見るよ」「この靴、あのジャンルで人気だよ」「このブランドあのダンサーさんも使ってたよ」などがあれば、下の報告欄から教えてください。確認しながら、適宜追加していきます。

私はストリートダンサーがよりよいダンス生活を送れることを願っています。

他にも「ジャンル別のおすすめ曲まとめ」などこんなサイトがあればいいななどありましたら、ご報告ください。

01 / GENRE

HipHop

HipHopは、ストリートダンスの中で最もポピュラーなジャンルです。 HipHopミュージックに合わせて、アップとダウンのリズムを取りながら様々なステップを踏んで踊ります。

    種別:靴のメーカー / 対象:ユニセックス

    まず最初に紹介するのはこちら、Onitsuka Tigerです。中でもOnitsuka TigerのGSM SDシリーズは薄目のソールが特徴で地面を感じやすい方が踊りやすいという方に向いています。カラーのバリエーションが多くとてもかわいいものも多いのでおすすめです。

    着用ダンサー例: AYUMI(D.LEAGUE / CHANGE RAPTURES)

    公式サイト ↗

  • Vans

    種別:靴のメーカー / 対象:ユニセックス

    ダンサーの足元といえば王道の VANS ですよね。その中でも、VANS の LITE シリーズはとにかく軽く人気です。 長時間ダンスする方におすすめで、ソールもとても滑りやすく踊りやすくなっています。 お手頃な価格なので、すぐに靴をぼろぼろにしてしまう方にもおすすめです。

    着用ダンサー例: Haruto(D.LEAGUE / CHANGE RAPTURES)

02 / GENRE

Lock

ロックダンスは1969年にDon Campbell（ドン・キャンベル）が考案したダンスです。 「Lock」とは「鍵をかける」という意味で、激しい動きから突然静止（ロック）するスタイルが特徴です。

    種別:衣類のメーカー / 対象:レディース向け

    Lockのレディース向けに紹介するブランドはこちら、PRANK PROJECT。とてもシルエットがかっこいい商品が多くダンスを踊るときにめっちゃいいじゃんとなる服が多いこと間違いなしです。普段着としてもいい服ばかりなのでぜひチェックしてみてください。

03 / GENRE

Pop

Popは1960〜70年代に形成されたダンスで、筋肉を弾く（POP）動きが特徴です。 ロボットダンスやムーンウォークなど、「人間らしくない不思議な動き」が魅力です。

  • tk.TAKEO KIKUCHI

    種別:衣類のメーカー / 対象:ユニセックス

    今回紹介するのはこちらtk.TAKEO KIKUCHI。テロッとした薄手な生地。だけどしっかりしてて高級感もありつつ踊りもしっかりみせやすいというPopperには最適な服ではないでしょうか。シャツもスラックスもあるのでよかったらチェックしてみてください。

    着用ダンサー例: NAOTO(D.LEAGUE / M&A SOUKEN QUANTS)

    公式サイト ↗

  • Super Cali

    種別:靴のメーカー / 対象:ユニセックス

    こちらは中国のブランドです。ダンサー用に作られたシューズで、他のシューズに比べてソールが少し薄いのが特徴です。思っているより柔らかいためすぐに自分の体になじむと思います。なんとpopping・boogalooを世界に広めたレジェンドダンスクルーであるElectric Boogaloosとのコラボ商品を出しています。これは買うしかないですね。

    着用ダンサー例: Shuru(ギャルとメイド) / xize(china)

    公式サイト ↗

  • Hush Puppies

    種別:靴のメーカー / 対象:ユニセックス

    Popperといえばこれ！Hush Puppiesですよね。有名なPopperほど履いてる勝手なイメージがあります。水色や黄色などかわいらしい色があるのが特徴的ですよね。きれいな色彩でこれを履いていれば目立つし周りと差別化することができます。勝負靴なんかにおすすめだと思います。ソールなどが壊れた時に店舗にいくと修理してもらえるそうです。ありがたいですね。

    着用ダンサー例: 勇太(龍と勇太) / Shuru(ギャルとメイド)

08 / GENRE

R&B

HipHopという大きな括りの中で、R&B HipHopの最大の特徴はR&Bの音楽に合わせて踊ることです。 HipHopの代名詞ともいえる「ラップ」がなく、スローなテンポに合わせて踊ることが特徴的なジャンルです。 また、通常ダンスは音楽のリズムやビートに合わせて踊りますが、R&B HipHopの場合は、使用する楽曲の歌詞に意識を向け、その歌詞の世界観を表現しようとする振り付けで踊ります。 そのため、アーティストのミュージックビデオのバックダンサーが踊っていることが多いです。

  • HARUTA

    種別:靴のメーカー / 対象:ユニセックス

    お次に紹介するのはこちら、HARUTAです。足になじむのが早くつま先の方は地面をキャッチできるようなっています。体重がいまこっちのっているななど感覚としてつかみやすいです。このローファーはどんな服でもしまってくれてオールジャンル似合う靴です。ぜひチェックしてみてください。

    着用ダンサー例: Keita

    種別:靴・衣類のメーカー / 対象:ユニセックス

09 / GENRE

Free Style

Free Styleは、さまざまなジャンルのダンスを特定せずに、多彩な技やステップを駆使して即興で踊るダンスです。 明確な定義はないため、たとえばHipHopとLockの2種類を取り入れたダンサーが、Free Styleを名乗ることもあります。 また多くのジャンルを取り入れることで、Free Styleを名乗るダンサーや、一つのジャンルのダンスをベースに、自分たちなりの解釈を加えてFree Styleと呼んでいる場合などがあります。

  • PRIME MEMBERS

    種別:衣類のメーカー / 対象:レディース寄り

    PRIME MEMBERSは韓国のブランドで主にレディース向けの衣装ブランドです。シルエットもよくて生地もダンスが映えるようになっている。踊りやすくセットアップなどいろんな柄があり種類豊富です。公式サイトも日本語対応でとても分かりやすいので気になった方はぜひ。

    着用ダンサー例: Nina Neves

  • NIKE

    種別:靴・衣類のメーカー / 対象:ユニセックス

    これまたもう何度目の登場かわからないですね、NIKEです。写真で紹介しているのはCOURT VISIONというシリーズでとにかく軽いのが特徴です。ソールも薄くこんなに踊りやすい靴あるんだとびっくりすることでしょう。1足持っているだけでなんのイベントでも使えると思うのでおすすめです。Shox R4というシリーズもいざ履いてみるとバネがありとても踊りやすいみたいです。
