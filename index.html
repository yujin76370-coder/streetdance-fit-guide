STREETDANCE FIT GUIDE

ストリートダンサー向けファッションブランドまとめ

このサイトは、ストリートダンサー向けのファッションブランドをジャンル別にまとめたガイドです。Lock や HipHop、Pop など、ジャンルごとにおすすめのブランドを紹介します。

オンラインで買えるブランドしか紹介していないので、気になったブランドは、ぜひ公式サイトから最新ラインナップをチェックしてみてください。

はじめに

うまいダンサーを見ていると、「あの人の服かっこいいな」「Lockダンサーがよく履いているあの靴、どこのだろう？」と思うことがあります。

でも、いざネットで調べてみても、ストリートダンスとファッションをジャンルごとにまとめている情報はまだ少なく、知りたい答えにたどり着けないことが多くありました。

それなら、自分で調べてまとめてみよう。
実際に履いている人に聞いたり、人づてに教えてもらったり、Instagramにあるストリートダンサーのインタビュー動画などを参考にしながら、ダンサー目線でブランドを集めていく。

そんなきっかけで始めたのが、STREETDANCE FIT GUIDE です。

このサイトでは、Lock、House、Pop、Break、HipHop、Waack、Krumpなど、ジャンルごとにブランドやアイテムを紹介していきます。

LockとPopはお互いのブランドをチェックすることを強くお勧めします。この2つのジャンル向けに商品を作っているブランドが多いので要チェックです。

まだ未完成な部分もたくさんありますが、これから少しずつ更新していく予定です。

「このブランドもよく見るよ」「この靴、あのジャンルで人気だよ」「このブランドあのダンサーさんも使ってたよ」などがあれば、下の報告欄から教えてください。確認しながら、適宜追加していきます。

私はストリートダンサーがよりよいダンス生活を送れることを願っています。

他にも「ジャンル別のおすすめ曲まとめ」などこんなサイトがあればいいななどありましたら、ご報告ください。

各ブランドの「着用ダンサー例」のお名前はクリック(タップ)できるようになっていて、押すとそのダンサーのInstagramプロフィールに飛べます。気になったダンサーがいたら、ぜひフォローして応援してみてください。

01 / GENRE

HipHop

HipHopは、ストリートダンスの中で最もポピュラーなジャンルです。 HipHopミュージックに合わせて、アップとダウンのリズムを取りながら様々なステップを踏んで踊ります。

  • NIKE

    種別:靴・衣類のメーカー / 対象:ユニセックス

    定番ブランドのNIKEです。中でも「AIR FORCE」は、分厚いソールが特徴で、重厚感のあるHipHopらしいスタイルを表現できます。特にブラウンカラーは、より重たく見えるためおすすめです。

    分厚いシューズが苦手な方には、「P-6000」がおすすめです。厚すぎず薄すぎないバランスの良いモデルとなっています。

    また、ソールが硬めで、つま先立ちなどを安定させたい方には「Air Jordan 37 Bordeaux」もおすすめです。靴裏のカラー配置がおしゃれで、踊っている際にチラッと見えることで存在感を演出できます。

    「AIR MAX PLUS」は、クッション性と安定性を兼ね備えた万能なシューズです。「Shox R4」シリーズもバネ感があり、とても踊りやすいと人気があります。

    着用ダンサー例: Moe Harukawa / Hinata(D.LEAGUE / SEGA SAMMY LUX) / TAKI(D.LEAGUE / SEGA SAMMY LUX leader) / MASATO / Luuny muuny / AK＠RI

    公式サイト ↗

  • Onitsuka Tiger

    種別:靴のメーカー / 対象:ユニセックス

    次に紹介するのはこちら、Onitsuka Tigerです。中でも「GSM SD」シリーズは、薄めのソールが特徴で、地面の感覚を感じやすい方が踊りやすいという人に向いています。

    カラーバリエーションも豊富で、かわいらしいデザインが多いのも魅力です。

    着用ダンサー例: AYUMI(D.LEAGUE / CHANGE RAPTURES)

    公式サイト ↗

  • Vans

    種別:靴のメーカー / 対象:ユニセックス

    ダンサーの足元といえば、やはり王道のVANSですよね。

    中でも「LITE」シリーズは、とにかく軽いことで人気があります。長時間踊る方におすすめで、ソールも滑りやすく、非常に踊りやすい仕様になっています。

    価格も比較的手頃なので、靴をすぐ傷めてしまう方にもおすすめです。

    着用ダンサー例: Haruto(D.LEAGUE / CHANGE RAPTURES)

    公式サイト ↗

  • CONVERSE

    種別:靴のメーカー / 対象:ユニセックス

    こちらも有名ブランド、CONVERSEです。

    今回紹介する「ALL STAR LIGHT OX」は、とにかく軽量なのが特徴のシューズです。かかと部分のソール裏がゴムではないため滑りやすく、厚みも少ないので、素足に近い感覚で床を感じることができます。

    床をしっかりつかみたい方におすすめの一足です。

    着用ダンサー例: KAI→(D.LEAGUE / CyberAgent Legit)

    公式サイト ↗

  • Dr.Martens

    種別:靴のメーカー / 対象:ユニセックス

    Dr. Martensのタッセルローファーは、とてもかっこいい一足です。

    見た目だけでなく機能性も高く、意外にもソールが柔らかいため、クッション性があります。滑りやすさと止まりやすさを兼ね備えた万能シューズです。

    普段履きしていくことで徐々に足になじみ、より踊りやすくなります。学生は学割で5％オフになるため、特におすすめです。

    着用ダンサー例: KANAU

    公式サイト ↗

  • DAF

    種別:衣類のメーカー / 対象:ユニセックス

    紹介するのはこちら、DAFです。

    HINATA.M がオーナーを務めるブランドで、「唯一無二」という思いが表れたアイテムがそろっています。

    デニムやセットアップなど、おしゃれなアイテムが豊富なので、ぜひチェックしてみてください。

    着用ダンサー例: HINATA.M(D.LEAGUE / KADOKAWA DREAMS)

    公式サイト ↗

  • MARINE SERRE

    種別:衣類のメーカー / 対象:ユニセックス

    次に紹介するのはこちら、MARINE SERREです。

    タイトめなトップスが多いブランドですが、写真のように、かわいらしいデザインが施されたスウェットなども展開しています。

    メンズアイテムもあるため、ぜひチェックしてみてください。

    着用ダンサー例: Moe Harukawa

    キャミソールや柄物も豊富で、とても使いやすいブランドです。

    最後に紹介するのは、レディース向けのMUSINSAです。

    日本からではなかなか見られない海外ブランドをまとめてチェックできるのが魅力です。かわいいアイテムが多く、価格帯も比較的手に取りやすい印象があります。

    ぜひ一度のぞいてみてください。

    着用ダンサー例: Moe Harukawa

02 / GENRE

Lock

ロックダンスは1969年にDon Campbell（ドン・キャンベル）が考案したダンスです。 「Lock」とは「鍵をかける」という意味で、激しい動きから突然静止（ロック）するスタイルが特徴です。

  • ELEFUNK

    種別:靴・衣類のメーカー / 対象:ユニセックス

    きました、みんな大好きELEFUNK。

    LockerやPopperといえばこの靴、と言わんばかりの定番ブランドです。このシューズは、Locker・Popper向けに作られています。

    エナメル素材を使用しており、革靴のような見た目ながら非常に柔らかく、履けば履くほど足になじみ、自分だけの一足になっていきます。

    カラーバリエーションや種類も豊富なので、ぜひチェックしてみてください。

    着用ダンサー例: REI(D.LEAGUE / dip BATTLES)

    公式サイト ↗

  • DIVERSITY

    種別:靴のメーカー / 対象:ユニセックス

    次に紹介するのはこちら、DIVERSITYです。

    実力派Lockerが多い韓国発のブランドで、シューズの種類やカラー展開が非常に豊富です。

    他ブランドにはないような奇抜なカラーも多く、「目立ちたい！」というLockerにはぴったりです。InstagramのDMから注文できるようなので、ぜひチェックしてみてください。

    公式サイト ↗

  • AmiAmi

    種別:靴・衣類のメーカー / 対象:レディース向け

    次に紹介するのは、レディース向けブランドAmiAmiです。

    画像の商品はレースアップシューズで、本格的な革靴ではなくフェイクレザー仕様のため、最初から履きやすく、裸足で踊っているように感じるほど軽量です。

    見た目もかわいく、どんなスタイルにも合わせやすい一足です。価格も比較的手頃なので、手に取りやすいと思います。

    アプリダウンロードで7％オフクーポンがあるため、ぜひチェックしてみてください。

    着用ダンサー例: Riho(D.LEAGUE / Medical Concierge I'moon)

    公式サイト ↗

  • ARGTFUNK!

    種別:衣類のメーカー / 対象:ユニセックス

    次に紹介するのはこちら、ARGTFUNK!です。

    名前からもFUNK感あふれる、インパクトのあるブランドです。「ありがたファンク！ 魂を揺さぶれる瞬間」をテーマに、FUNK愛が詰まった商品がそろっています。

    服やアイテムにもFUNKの世界観が表現されており、FUNK好きにはたまらないブランドです。ぜひ勝負服に取り入れてみてください。

    公式サイト ↗

  • FUNKOLOGIC

    種別:靴・衣類のメーカー / 対象:ユニセックス

    次に紹介するのはこちら、FUNKOLOGICです。

    こちらもFUNKの名を冠した、ワクワクするブランドですね。

    FUNKOLOGICは、LockerやPopper向けの商品を展開しており、世界44か国で展開されています。

    シューズだけでなく、ストライプパンツやジャケット、キャップなども販売しているので、ぜひチェックしてみてください。InstagramのDMから注文可能です。

    公式サイト ↗

  • PIN NAP

    種別:衣類のメーカー / 対象:レディース向け

    次に紹介するのは、原宿にあるレディース向けブランド「PIN NAP」です。

    ギャル系の雰囲気が特徴で、個性的な洋服が多く、バトルで目立ちたい方や勝負服を探している方にぴったりです。

    ジャケットなどアイテム数も豊富なので、ぜひチェックしてみてください。

    着用ダンサー例: Riho(D.LEAGUE / Medical Concierge I'moon)

    公式サイト ↗

  • PRANK PROJECT

    種別:衣類のメーカー / 対象:レディース向け

    次に紹介するのはこちら、PRANK PROJECTです。

    シルエットが非常にかっこいいレディース向けアイテムが多く、「踊ったときに映える！」と思える服がそろっています。

    普段着としても使いやすいアイテムが多いので、ぜひチェックしてみてください。

    着用ダンサー例: ena(CyberAgent Legit)

    公式サイト ↗

  • 政吉

    種別:衣類のメーカー / 対象:ユニセックス

    ロックダンサー向けとしては珍しい、衣装ブランド「政吉」です。

    LockやPop向けの靴ブランドは多い印象ですが、衣装ブランドは少ないため、とても貴重な存在です。

    生地の質が高く耐久性にも優れているため、ゲッダンや開脚など激しい動きをしても破れにくいのが特徴です。

    また、特殊な作りになっており、一般的なズボンより足が上がりやすく、シルエットも非常にかっこよく見えます。

    Yahoo!ショッピングでも販売されていますが、在庫切れが多いため、InstagramのDMからオーダーするのがおすすめです。

    着用ダンサー例: TAKARA / JJ

    公式サイト ↗

03 / GENRE

Pop

Popは1960〜70年代に形成されたダンスで、筋肉を弾く（POP）動きが特徴です。 ロボットダンスやムーンウォークなど、「人間らしくない不思議な動き」が魅力です。

  • Hush Puppies

    種別:靴のメーカー / 対象:ユニセックス

    Popperといえばこれ、Hush Puppiesですよね。

    有名なPopperほど履いているイメージがある定番シューズです。水色や黄色など、かわいらしいカラー展開が特徴的です。

    鮮やかな色合いで、履くだけで目立ち、周りとの差別化もできます。勝負靴としてもおすすめです。

    ソールなどが壊れた際は、店舗で修理対応してもらえるそうです。

    着用ダンサー例: 勇太(龍と勇太) / Shuru(ギャルとメイド)

    公式サイト ↗

  • Super Cali

    種別:靴のメーカー / 対象:ユニセックス

    こちらは中国発のブランドです。

    ダンサー向けに作られたシューズで、他のシューズと比べてソールがやや薄めなのが特徴です。

    見た目以上に柔らかく、すぐに足になじみやすいシューズとなっています。

    さらに、Popping・Boogalooを世界に広めた伝説的ダンスクルー Electric Boogaloos とのコラボ商品も展開しています。

    着用ダンサー例: Shuru(ギャルとメイド) / xize(china)

    公式サイト ↗

  • Blance

    種別:靴・衣類のメーカー / 対象:ユニセックス

    こちらもPopperに人気のブランドです。

    モデルの KITE さんの出身地である Los Angeles のコアカルチャーを追求したブランドとなっています。

    シルエットが非常にきれいで、カラー展開も豊富です。履けば履くほど足になじんでいきます。

    ストライプパンツなど、シューズ以外のアイテムも展開しているので、ぜひチェックしてみてください。

    着用ダンサー例: RIRIA(D.LEAGUE / M&A SOUKEN QUANTS) / KITE(Former Action) / ena(CyberAgent Legit)

    公式サイト ↗

  • PRO-Keds

    種別:靴のメーカー / 対象:ユニセックス

    1949年創業のアメリカ発ブランド「PRO-Keds」です。

    「ROYAL AMERICA」シリーズは、シルエットがCONVERSEに似ていますが、CONVERSEよりもソールがやや柔らかく、つま先立ちなどを安定させたい方におすすめです。

    着用ダンサー例: NAOTO(D.LEAGUE / M&A SOUKEN QUANTS)

    公式サイト ↗

  • Reebok

    種別:靴のメーカー / 対象:ユニセックス

    次に紹介するのはReebokです。

    特に「PREMIER ROAD PLUS VI」は、見た目に反して非常に軽く、デザインもかわいらしいモデルです。

    横幅が広めのため、足が大きい方や偏平足の方でも痛くなりにくく、快適に踊ることができます。

    着用ダンサー例: KISA(D.LEAGUE / KADOKAWA DREAMS)

    公式サイト ↗

  • tk.TAKEO KIKUCHI

    種別:衣類のメーカー / 対象:ユニセックス

    今回紹介するのはこちら、tk.TAKEO KIKUCHIです。

    テロッとした薄手の生地感ながら、高級感があり、動きもしっかり見せやすいアイテムが多いため、Popperにはぴったりのブランドではないでしょうか。

    シャツやスラックスも展開されているので、ぜひチェックしてみてください。

    着用ダンサー例: NAOTO(D.LEAGUE / M&A SOUKEN QUANTS)

    公式サイト ↗

  • ワカモレ

    種別:衣類・グッズのメーカー / 対象:ユニセックス

    こちらは Former Action の KITE さんがオーナーを務めるブランド「ワカモレ」です。

    KITEさんが主催・プロデュースに深く関わっているダンスイベント POPBOXX のオフィシャルグッズ販売など、さまざまなアイテムを展開しています。

    KITEさんファンにはたまらないブランドですね。価格帯も比較的手頃です。

    着用ダンサー例: KITE(Former Action)

04 / GENRE

House

Houseは、シカゴやニューヨークのクラブで生まれたダンスです。 Houseミュージックという4つ打ちのアップテンポな音楽に合わせて、軽快なステップを踏むのが特徴です。

  • adidas

    PHOTO COMING SOON

    種別:靴のメーカー / 対象:ユニセックス

    最初に紹介するのは、王道ブランドのadidasです。

    中でも「SAMBA」「SPEZIAL」「SUPERSTAR」は、Houserから定番の支持を集めているモデルです。いずれもローカットで足首が動かしやすく、Houseの軽やかなフットワークを邪魔しません。

    ほどよく重みがあるので足元が安定しやすく、初めてのダンスシューズとしてもおすすめです。アスファルトのように靴裏の消耗が激しい場所で踊ることが多い方は、ラバー底のモデルを選ぶと安心です。

    公式サイト ↗

  • VANS

    種別:靴のメーカー / 対象:ユニセックス

    HipHopでも紹介した、王道ブランドVANSです。

    Houserにとっても足元は非常に重要ですが、やはりVANSは人気があります。中でも「Authentic」「Old Skool」「LITE」シリーズは、それぞれフィット感やソールの厚みが異なるので、自分の好みに合わせて選べます。

    ローカットで足首の動きを邪魔せず、長時間踊る方にも、靴をすぐ傷めてしまう方にもおすすめです。価格も手頃なので、最初の一足として選びやすい一品です。

    公式サイト ↗

  • PUMA

    PHOTO COMING SOON

    種別:靴のメーカー / 対象:ユニセックス

    次に紹介するのはPUMAです。

    中でも「PUMA Suede」「Speedcat」はHouserに愛されるロングセラーモデルです。スウェード素材で柔らかく、履けば履くほど足になじんでいきます。

    Speedcatは特に薄底でフロアの感覚を取りやすく、ステップを正確に踏みたい方に向いています。スタジオやステージなど、すこし滑りクッション性のある場所でもしっかりフィットして安定感のある一足です。

    着用ダンサー例: MEI(Novel Nextus)

    公式サイト ↗

05 / GENRE

Waack

Waackは1970年代にゲイクラブやディスコで生まれたダンスです。 腕を鞭のようにしなやかに振り回す動きが特徴で、女性らしいポージングも魅力です。

  • AmiAmi

    種別:靴・衣類のメーカー / 対象:レディース向け

    再び登場、レディース向けブランドAmiAmiです。

    Lockでも紹介しましたが、Waackerにもおすすめのため、改めて紹介します。

    画像の商品はレースアップシューズで、フェイクレザー仕様のため最初から履きやすく、裸足感覚で踊れるほど軽量です。

    見た目もかわいく、どんなスタイルにも合わせやすい一足です。

    価格も比較的手頃で、アプリダウンロードによる7％オフクーポンもあるため、ぜひチェックしてみてください。

    着用ダンサー例: Riho(D.LEAGUE / Medical Concierge I'moon)

    公式サイト ↗

06 / GENRE

Break

Breakは1970年代にニューヨーク・サウスブロンクスで生まれたダンスです。 2024年パリオリンピックで「ブレイキン」として正式種目に採用され、世界的に注目を集めています。

  • PRO-Keds

    種別:靴のメーカー / 対象:ユニセックス

    Popでも紹介した「PRO-Keds」です。

    ゴツめのシューズではなく、スラッとしたフラットな形状が特徴です。

    BreakでもPop同様、「ROYAL AMERICA」シリーズが人気で、地面との距離感を近く感じられるため、ステップを踏みやすい・踏ん張りやすいと感じる方におすすめです。

    着用ダンサー例: REIMI(D.LEAGUE / KOSÉ 8ROCKS)

    公式サイト ↗

  • SALOMON
